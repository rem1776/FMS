Detailed Description

This module provides interfaces to the non-domain-oriented communication subroutines.

Mainly includes interfaces and type definitions for reproducing operations with extended fixed point data.

mpp_reproducing_sum Interface

This interface uses a conversion to an integer representation of real numbers to give order-invariant sums that will reproduce across PE count.This idea comes from R. Hallberg and A. Adcroft.

27 !> @brief
28 !> A convenience file to use any FMS routines, functions, values
29 !>
30 !> Imports all supported FMS modules so that any public interfaces,
31 !! variables or routines can be used via this module. Excludes mpp_io modules
32 !! and routines. Overloaded type operators/assignments cannot be imported individually
33 !! (ie. `use fms, only: OPERATOR(*)` includes any defined '*' operators within FMS).
34 !!
35 !! Remappings due to conflicts:
36 !!
37 !! get_mosaic_tile_grid from mosaic2(fms2_io) => mosaic2_get_mosaic_tile_grid
38 !!
39 !! read_data from interpolator_mod(fms2_io) => interpolator_read_data
40 !!
41 !! ZERO from interpolator_mod(mpp_parameter) => INTERPOLATOR_ZERO
42 !!
43 !! Not in this module:
44 !!
45 !! axis_utils_mod, fms_io_mod, time_interp_external_mod
46 !! get_grid_version_mpp_mod, mpp_io_mod, mosaic_mod,
47 !! fms_mod(partial, old io excluded), drifters modules
48 !!
49 !! A full list of supported interfaces and public types intended for use via
50 !! this module is provided in the [supported_interfaces.md](../../supported_interfaces.md)
51 !! file.
